# 🎉 PROJECT COMPLETED - CANDIDATE APPLICATION & ADMIN DASHBOARD

## 📦 What's Been Created

A complete, production-ready **Candidate Application & Admin Dashboard System** with:
- ✅ Full Node.js + Express backend
- ✅ MongoDB database integration
- ✅ EJS server-side rendering (no HTML files)
- ✅ File upload with Multer
- ✅ Admin authentication with sessions
- ✅ Responsive CSS styling
- ✅ Clean MVC architecture

## 🚀 QUICK START (3 STEPS)

### 1. Install Dependencies
```bash
cd C:\Users\Lenovo\Desktop\20k
npm install
```

### 2. Start MongoDB
```bash
mongod
```
(Keep this running in a separate terminal)

### 3. Run the Server
```bash
npm run dev
```

**Server will start at:** `http://localhost:3000`

## 🛠️ TROUBLESHOOTING

### Issue: Can't connect to MongoDB
**Solution:** Make sure MongoDB is running (`mongod` command)

### Issue: Port 3000 already in use
**Solution:** Change PORT in .env file or kill process using port 3000

### Issue: File upload fails
**Solution:** File must be PDF/DOC/DOCX and less than 5MB

### Issue: Admin login doesn't work
**Solution:** Clear browser cookies and check .env credentials
